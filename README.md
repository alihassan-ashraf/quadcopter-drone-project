# 📖 Project Overview

This project presents the complete development of an **F450 Quadcopter Drone** built as part of the **Introduction to Robotics** course at the **Department of Information and Communication Engineering, The Islamia University of Bahawalpur**.

Unlike many drone projects that only show the final product, this repository documents the **entire engineering workflow**, beginning from component selection and hardware assembly to software configuration, calibration, flight testing, troubleshooting, crash analysis, repair, and successful re-flight.

The project provided practical experience in robotics, embedded systems, UAV technology, flight control, electronics integration, and system debugging.

---

# 🎯 Objectives

- Build a fully functional quadcopter from scratch
- Understand UAV architecture
- Learn Pixhawk flight controller configuration
- Configure Mission Planner
- Perform complete sensor calibration
- Understand ESC and motor synchronization
- Test real outdoor flight
- Analyze failures and improve system reliability

Software Configuration

- Accelerometer Calibration

---

## Phase 5

Flight Testing

- Pre-flight inspection
- Safety verification
- First hover
- Outdoor testing
- Flight stability evaluation

---

## Phase 6

Failure Analysis

During the first outdoor flight the drone crashed after losing stable control.

Damage included:

- Broken propellers
- Damaged frame arm

The root cause was investigated, damaged components were replaced, and the drone successfully returned to flight.

This project demonstrates the importance of systematic troubleshooting and iterative engineering design.
